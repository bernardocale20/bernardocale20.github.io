Templates de Site feitas com HTML e CSS.

Descrição:

Quando iniciei meu aprendizado em desenvolvimento web alguns meses atras, um dos meus primeiros projetos consistia fazer sites fakes, ou "mockups", cujo objetivo era treinar a formatação de uma pagina, estilização dos templates, dentre outros, assim mostrando como o processo de criação de um site pode ser relativamente simples a nivel básico.

Um dos primeiros que eu fiz foi a criação de um website de uma empresa fake de viagens, no caso tudo foi inventado, o motivo era testar e compreender diferentes uso das tags do html tanto as antigas quanto as novas que chegaram com o html 5 e utilizar do css como meio de formatação externo aos elementos da página.

Ao todo varios sites foram feitos, desde a sites que tentavam recriar modelos de blog, ate mesmo sites de notícias, eu deixei salvo em link abaixo dois deles, e para melhor entendimento das funções html para aqueles que estão iniciando como desenvolvedores web e que, como maneira ideal de aprendizado, gostam de analisar o design de diversos sites e criar diversos modelos de template do zero, recomendo que instalem esse plugin para o seu navegador "Web Developer" , nele voce poderá alterar funções de exibição do navegador, manipulando a forma que o css é lido, dentre outros. As templates se encontram abaixo.
