Objetivo: Destrua todos os 104 asteroides antes que o tempo acabe.


Antes de começar, selecione o nivel de dificuldade, sao 3 ao todo: Fácil, Normal e Difícil.

Apos selecionar o nível de dificuldade clique no ícone Play! para começar a jogar.

Os niveis de dificuldade alteram o tempo limite para o término do jogo, então fique atento!


Projeto Asteroids: Feito com Html, CSS e JavaScript.

Obs: Para jogar este jogo de forma mais otimizada, jogue-o por meio de navegadores desktop.
